Chatroom Security Challenge

Challenge: implement an update security layer for messages using only allow & deny.

Messages can be edited (note that this is not implemented in the UI, so you can just do it through the console) and liked. You can only edit your own messages, but you can like anybody's messages once.

Liking should also update a likesCount property on the message object that holds the total number of likes.
